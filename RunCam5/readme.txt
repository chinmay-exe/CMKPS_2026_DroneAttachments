# RunCam 5 Orange Camera Holder

## Overview

This project contains a custom camera holder designed for the RunCam 5 Orange action camera. The holder was developed for a custom-built quadcopter to provide a secure mounting solution while reducing camera vibrations and jitter during operation.

The design consists of:

* Camera Mount
* Support Attachment
* Adjustable Pivot Connection

The parts were designed in Fusion 360 and manufactured using TPU through FDM 3D printing.

## Features

* Lightweight design
* Friction-fit camera mounting
* Additional zip-tie safety retention
* TPU vibration damping
* Adjustable camera angle
* Compatible with custom quadcopter frames

## Software Used

* Fusion 360 (CAD Design)

## Manufacturing

* Technology: FDM 3D Printing
* Material: TPU
* Printer: Creality 3D Printer

## Files Included

* `.f3d` Fusion 360 source files
* `.stl` files for 3D printing
* Project documentation
* Assembly images

## Camera Specifications

Designed for:

* RunCam 5 Orange

Camera mount internal dimensions:

* Length: 38.8 mm
* Width: 38.8 mm
* Height: 36.6 mm

## Assembly

1. Print all components using TPU.
2. Insert the RunCam 5 Orange into the camera mount.
3. Secure the camera using a zip tie if required.
4. Connect the camera mount to the support attachment using the pivot joint.
5. Attach the assembly to the quadcopter frame.
